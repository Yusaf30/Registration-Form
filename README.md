# HTML Registration Form

This project is a simple **Registration Form** created using **HTML only**.
The form collects user information such as full name, email address, password, gender, interests, experience level, favorite session, and personal details.

The purpose of this project is to practice **HTML form elements and basic web page structure**.

---

## Features

* User input fields for personal information
* Email and password fields
* Gender selection using radio buttons
* Interests selection using checkboxes
* Dropdown menu for experience level
* Dropdown menu for favorite session
* Text area for writing about yourself
* Submit button to send form data

---

## Technologies Used

* HTML5

---

## Project Purpose

This project was created while learning **HTML forms** and understanding how different input elements work in a webpage.

It helps in practicing:

* Form structure
* Input fields
* Radio buttons
* Checkboxes
* Dropdown menus
* Text areas
